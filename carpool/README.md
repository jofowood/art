# School Carpool Timer

A simple app to calculate when you need to leave for school carpool.

## How to Host on GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [GitHub](https://github.com) and sign in
2. Click the **+** button in the top right, then **New repository**
3. Name it something like `carpool-timer`
4. Make it **Public**
5. Click **Create repository**

### Step 2: Upload Your Files
1. On your new repository page, click **uploading an existing file**
2. Drag and drop the `index.html` file
3. Click **Commit changes**

### Step 3: Enable GitHub Pages
1. In your repository, click **Settings** (top menu)
2. Click **Pages** in the left sidebar
3. Under "Source", select **main** branch
4. Click **Save**
5. Wait a minute, then refresh the page
6. You'll see a message with your site URL: `https://[your-username].github.io/carpool-timer/`

### That's it!
Your carpool timer will be live at that URL. You can bookmark it on your phone for easy morning access.

## Using the App
- Set your arrival time (defaults to 8:45 AM)
- Check/uncheck which kids need rides
- Check/uncheck if you need to scrape ice
- The app calculates when you need to start!
